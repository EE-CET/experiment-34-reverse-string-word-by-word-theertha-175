# Experiment 34: Reverse String Word by Word

## Problem Statement

Write a program to reverse the string word by word.

**Note:** The words should be separated by a single space in the output. Leading or trailing spaces should be removed.

## Input Format

* A single sentence string.

## Output Format

* The reversed words string.

### Example 1

**Input:**

```text
Hello World Java
```

**Output:**

```text
Java World Hello
```

### Explanation
The input sentence is split into "Hello", "World", "Java". Reversing their order gives "Java", "World", "Hello".
